 <h4>주요기능</h4>
 <ul class="square">
   <li>회원가입, 로그인</li>
   <li>상품검색, 상품조회, 장바구니, 결제</li>
   <li>관리자 ( 상품, 회원, 주문관리 )</li>
   <li>게시판 ( Q&A게시판, 리뷰게시판 )</li>
   <li>이메일인증(javax.mail), 간편결제 (Improt), 우편번호 (kakao) </li>
</ul>
<h4>프로젝트 후기</h4>
  <ul class="square">
   <li>팀원간의 협업을 통해 커뮤니케이션 능력 향상</li>
   <li>오픈API를 적용해보며 공식문서를 보고 검색하는 능력의 향상</li>
   <li>스프링 DI, IoC 등 스프링MVC의 대한 이해도 증가</li>
   <li>기본 비지니스 로직에 대한 이해도 증가</li>
   <li>오류를 만났을 때 극복해 나가며 문제해결능력 향상 </li>
</ul>
